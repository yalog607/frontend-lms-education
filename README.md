# 🎓 LMS Education System (MERN Stack)

A full-stack Learning Management System (LMS) built with MERN Stack, supporting course management, user roles, and authentication.

---

## 🚀 Features

- 👨‍🏫 Course management (create, update, manage courses)
- 👥 User authentication & role-based access (Admin / Student)
- 🔐 Secure authentication using JWT
- 📦 RESTful API architecture
- 🎨 Responsive frontend with React
- ⚡ Optimized database queries

---

## 🛠 Tech Stack

**Frontend:** React.js, Axios, CSS  
**Backend:** Node.js, Express.js, MongoDB, JWT  
**Deployment:** AWS EC2, Nginx, PM2  

---

## 🏗 System Architecture

Client (React) → Backend API (Node.js + Express) → Database (MongoDB)

---
